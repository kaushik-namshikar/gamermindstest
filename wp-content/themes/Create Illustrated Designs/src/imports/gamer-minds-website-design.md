Design a modern, premium, conversion-focused website for Gamer Minds — Game Localization Collective.

This is a two-path website with a soft gate landing page that routes users to:

• /developers (B2B localization services)
• /players (community Discord voting platform)

The website should feel:
• Modern
• Professional but creative
• Gaming-industry aligned
• Trustworthy for studios
• Slightly playful on player side

Use a dark UI base with subtle gradients (purple/indigo tones inspired by current branding), clean typography, and strong CTA buttons.

Design Desktop first (1440px width), then Mobile (390px).
