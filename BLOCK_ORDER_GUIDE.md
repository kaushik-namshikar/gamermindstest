# GamerMinds Block Order Guide

This document outlines the recommended block order for each page template in the GamerMinds WordPress theme.

---

## 1. Front Page (Homepage)

The homepage should showcase your services and convert visitors. Recommended block order:

1. **gm/hero** - Main hero section with compelling headline
   - Title: "Professional Game Localization Services"
   - Subtitle: "Reach global audiences in 50+ languages"
   - CTA button linking to contact

2. **gm/services-grid** - Showcase main service offerings
   - Game Localization
   - Voice Acting
   - Translation Services
   - Testing & QA

3. **gm/trust-grid** - Build credibility with trust indicators
   - Years in business
   - Games localized
   - Languages supported
   - Team members

4. **gm/language-grid** - Display language capabilities
   - Show supported languages organized by region
   - Include native speaker count per language

5. **gm/case-grid** - Portfolio/case studies
   - 3-6 featured game localization projects
   - Show impact metrics (languages, regions, etc.)

6. **gm/process-timeline** - Show your localization process
   - Planning phase
   - Translation phase
   - Voice acting phase
   - QA & testing phase
   - Delivery phase

7. **gm/quote-form** - Call to action for quotes
   - Lead capture form
   - Ask about project details

8. **gm/faq-accordion** - Answer common questions
   - Timeline questions
   - Language availability
   - Pricing & costs
   - Technical requirements

9. **gm/cta** - Final call to action
   - Motivational closing message
   - Button to contact/get started

## 2. Developers Page

5. **gm/process-timeline** - Tech-focused workflow
   - Setup & integration
   - Content extraction
   - TMS workflow
   - Build & testing

6. **gm/case-grid** - Developer case studies
   - Major studios worked with

   - Version support

   - Join developer network
   - Access technical resources

## 4. Generic Inner Pages (Blog, Services, etc.)

For additional pages not using specific templates:

1. **gm/hero** - Page-specific hero
   - Page title and description

2. **gm/split-cards** or **gm/services-grid** - Main content
   - Break up content into digestible blocks

3. **gm/process-timeline** or **gm/case-grid** - Supporting content
   - Show relevant work/expertise

4. **gm/faq-accordion** - Topic-specific FAQ

5. **gm/quote-form** - Call to action
   - Lead capture

6. **gm/cta** - Final engagement
   - Next step button

## Block Composition Rules

### Do's ✅
- Start every page with **gm/hero** for context
- Use **gm/split-cards** for 2-3 point comparisons
- Use **gm/services-grid** for 4+ service listings
- End pages with **gm/quote-form** + **gm/cta** for conversion
- Alternate between content blocks and visual breaks
- Use spacing between major sections

### Don'ts ❌
- Don't use multiple heroes on same page
- Don't stack more than 2 grids back-to-back without content
- Don't use case-grid for less than 3 items
- Don't use CTA block more than once per page
- Don't forget call-to-action sections

---

## Responsive Considerations

All blocks are fully responsive and automatically adapt to:
- Mobile (< 480px)
- Tablet (480px - 768px)
- Desktop (> 768px)

No manual adjustments needed per device.

---

## Customization

In the WordPress Block Editor, you can:
1. Rearrange blocks by dragging
2. Edit block content with the inspector panel
3. Choose button styles, colors, and actions
4. Add custom CSS classes for styling

All customizations persist in the page/post content.
